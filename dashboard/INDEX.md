## ğŸ”§ Quick Reference

### Essential Commands
```bash
# Setup
cd dashboard && npm install

# Development
npm start          # Run dev server (http://localhost:3000)

# Production
npm run build      # Create optimized build
npm test           # Run tests
```

### File Locations
```
Dashboard main:     src/Dashboard.js
Components:         src/components/
API calls:         src/services/api.js
Styles:            src/*.css + components/*.css
Entry point:       src/index.js
HTML:              public/index.html
Config:            package.json
```

### Key Concepts
- **Components**: Reusable UI pieces
- **State**: Data that changes over time
- **Props**: Data passed to components
- **useEffect**: Run code when component mounts
- **useState**: Create stateful variables
- **API**: Communication with backend
